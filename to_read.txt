A ordem dos Registers são:

rax		system call number		This is the return value.
rdi		1st argument
rsi		2nd argument
rdx		3rd argument
r10		4th argument
r8		5th argument
r9		6th argument

Data Types:

Byte 				8 bits
Word 				16 bits
Double Word			32 bits
Quad Word			64 bits
Double Quad Word		128 bits
